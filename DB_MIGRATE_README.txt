DB Commands:
python manage.py db --help


Create auto migration to next version (detects from model updates):
python manage.py db migrate

Generate version file shell, to write migration from scratch:
python manage.py db revision


Update the script by hand if it was autogenerated and something isn't detected.

Run the actual migration to modify the database:
python manage.py db upgrade
